# 🤖 Chatbot LIM - Frequência

Chatbot simples em **Streamlit** para esclarecimento de dúvidas sobre regras de frequência dos colaboradores dos LIMs.  
As respostas são **fixas**, com opção de enviar e-mail à Área de Gestão de Pessoas caso a dúvida não seja resolvida.
